# Tuition Management System

A Spring Boot web application for managing tuition class students, batches, enrollments, attendance, and monthly payments.

## Features Completed

- Student Management
- Batch / Class Management
- Student Enrollment
- O/L and A/L class structure
- Grade 10, Grade 11, Grade 12, and Grade 13 support

## Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- Thymeleaf
- MySQL
- Bootstrap
